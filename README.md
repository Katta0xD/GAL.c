# GAL.c
Little C library and CLI tool for matrix manipulation, includes:
  * row operations
  * row reduction to row echelon form and reduced row echelon form
  * arithmetic operations
    * addition
    * scalar multiplication
    * matrix multiplication
  * calculation of
    * determinant
    * inverse
    * rank
    * transpose
    * diagonal matrix  

This is a learning project intended to delve into C and linear algebra, but feel free to use it if you find it any helpful.
